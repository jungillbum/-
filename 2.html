스킬선택

대검을 들기 위해서는 56레벨 각성 퀘스트 후 '각성'을 선택 해야 한다.
  • 각성스킬 : 모두 찍어야 한다. 무조껀!!!
  • 본스킬 : 취향이 많이 갈린다. 개인적인 추천은 작열!
    • 작열 : 진입기 보강용, 작열에 슈아(슈퍼아머 : 공격받아도 캔슬되지 않음) 추가
    • 광기부수기 : PVE용
    • 무덤가르기 : 상태이상 보강용, 마지막 타격 넉다운이 PVP에도 적용됨
  • 기본스킬 : 남은 스킬 포인트로 찍는다.
    • 패시브 : 체력 단련 제외 필수, 딱 보면 느낌이 온다! 뭐 부터 찍어야 할지.
    • 돌진찌르기 : 이동기, 꼭 배운다.
    • 잡아매치기 : 워리어 PVP의 핵심 스킬! 배운다.
    • 기타 : 방패진공, 전진검술은 이동기로 쓸모 있으니 먼저 찍는다. 나머지는 여유가 되면 하나씩 찍어 본다.
  • 기술심화 : 잡아매치기까지 찍고 남은 스킬 포인트로 찍는다.
    • 광기어린창 : PVE 이동기, PVP 진입기, 배우면 좋다.
    • 방패올려밀치 : PVP 가까운 거리 진입기, 배울까?
    • 기타 : 그닥....
  • 기술특화 : 생각보다 사냥 스피드, PVP에 영향을 끼친다. 신중하게 고려하자!
    1. 기본원칙 : 콤보 초반 -> 버프 특화, 콤보 후반 -> 피해 증가 특화
    2. 참고기술 : 오리지널 스킬 버프 참조하여 특화하는 것도 좋다.
    3. 주의사항 : 약+강=강 but 강+약=기존 강만 유지(시간 증가 없음)
    4. 추천특화 : 공속 7% 항시 유지, 다운스매쉬로 PVP 콤보 이어가기
