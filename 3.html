수정선택

죽는 비율에 따라 최고성능 -> 가성비 -> 저렴이로 가야 한다. 수정은 소모품이다.
  • 기본적인 세팅
    • 수정 세팅+음식으로 공속5단/이속5단은 되어야 한다. 치명도 5단이면 좋지만, 크게 의미 없다.(주력 스킬 치명 100%)
    • 펄옷 : 이속 X 1
    • 뚜껑 : 강훔 X 2
    • 갑옷 : 강훔 X 2
    • 장갑 : 적중/치명 X 2
    • 신발 : 금강 X 2
    • 무기 : 적중 X 2
    • 보조 : 치명 X 2
    • 단델 : 강훔 X 2, 갑옷을 원하는 것으로 변경
  • PVP 세팅
    • 가심이 없다면 PVP 세팅은 비추! (특회의 효율이 떨어진다)
    워리어는 적중 보정을 받는 스킬이 많아서 악세로 적중을 올릴 필요는 없지만 기본은 해야 한다.
    • 펄옷 : 이속 X 1
    • 뚜껑 : 강훔 X 2
    • 갑옷 : 특회 X 2
    • 장갑 : 적중/치명 X 2
    • 신발 : 금강/저항(자신이 약한 부분) X 2
    • 무기 : 적중 X 2
    • 보조 : 치명 X 2
    • 단델 : 강훔 X 2
모든 특화와 수정은 자신의 콤보와 어울려야 한다. 정해진 답은 없다.
